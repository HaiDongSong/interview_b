##### 布局方案：

1. 浮动布局
2. 绝对定位
3. flex box
4. table布局
5. 网格布局

##### 这几种布局方式的优缺点

| layout | advantage | shortcoming | 超出高度能用吗 |
| :--- | :--- | :--- | :--- |
| 浮动布局 | 兼容性高 | 脱离文档流 | no |
| 绝对定位 | 兼容性高 | 脱离文档流 | no |
| flex布局 | 解决以上两种布局缺点 | 不兼容IE8 | yes |
| table布局 | 和flex差不多 | 麻烦 | yes |
| grid布局 | heflex差不多，代码简单 |  | no，需要设置高度 |
